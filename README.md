# gammaPAC

Python implementation of "A mutual information measure of phase-amplitude coupling using gamma generalized linear models":
https://www.frontiersin.org/articles/10.3389/fncom.2024.1392655/full 

This code repository provides the functions necessary to implement the methods found in the above citation. The critical functions to implement the gammaPAC method are found in gammaPAC.py, while other python files contain helper or utility functions
